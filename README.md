# FPGA True Random Number Generator

Аппаратный генератор псевдослучайных чисел на основе анализа шумов микрофона.
Реализован на ПЛИС Xilinx Artix-7 xc7a100tcsg324.

## Описание

Устройство использует встроенный PDM-микрофон ADMP421 как источник физической энтропии.
Шум обрабатывается через CIC-фильтры, спектральный анализ CORDIC и многоуровневое
кондиционирование (LFSR, CRC-32, Wang hash).

## Архитектура

- **Микрофон**: ADMP421 PDM @ 2.38 МГц
- **Фильтрация**: CIC 4-го порядка, децимация ×50
- **Спектральный анализ**: CORDIC DFT (5/9/13.5 кГц)
- **Извлечение энтропии**: XOR LSB + Von Neumann дебайсинг
- **Кондиционирование**: LFSR×2 + CRC-32 + Wang hash
- **Выход**: 32-битные числа @ ~1/30мс

## Статистика

| Тест | Результат | Порог |
|------|-----------|-------|

| Хэмминг | 15.2/32 | 12-20 |
